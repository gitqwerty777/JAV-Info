## usage

`python getInfo.py`

change the config in `config.json`

## config.json

| key  | description|
|------|------|
|fileExts | extension|
|fileDir | directory|
|getInfoInterval | interval, do not set too small|
|fileNameFormat | newfilename|
|language | tw, cn, en, ja|
|saveAlbum | tw, cn, en, ja|
